📖 User Guide

Welcome to the Simply Rugby Management System. This desktop application streamlines club operations.

Getting Started: Login using your assigned credentials. The Dashboard provides a live count of players and active injuries.
  • Member Directory: Add, edit, or remove players from the database.
  • Medical Records: Log injuries. Ensure you tick "Cleared to Play" once a player recovers.
  • Training Skills: Rate player performance across Passing, Tackling, and Kicking.

🛠️ Technical Reference

This section is for system administrators and academic examiners.

Database Architecture

The system utilizes Entity Framework Core with a local SQL database. Key models include:

  • Member: Primary entity for all club participants.
  • User: Security entity for authentication with BCrypt hashed passwords.
  • TrainingSession: Handles schedule logic and pitch allocation.

Security Implementation

Passwords are never stored in plain text. We implement the BCrypt.Net-Next library for one-way cryptographic hashing.

❓ FAQ

Q: Why is the Safeguarding list empty?
A: The system automatically filters players. Only those under 18 years of age based on their DOB will appear.

Q: How is the 'Injured' count calculated?
A: It counts all medical records where the IsRecovered status is set to false.
